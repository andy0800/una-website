Troubleshooting

Common Issues & Solutions:

  1. Can't access from other devices?
    • Check if devices are on the same WiFi network
    • Verify the server is running
    • Check Windows Firewall settings
  2. Connection refused?
    • Ensure server is listening on 0.0.0.0
    • Check if port 3000 is not blocked
  3. Slow loading?
    • Check network speed
    • Reduce image sizes if needed
