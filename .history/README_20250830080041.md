### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd una-institute-website
   ```

2. **Install dependencies**
   ```bash
   npm run install:all
   ```

3. **Environment setup**
   ```bash
   cp env.example .env
   # Edit .env with your configuration
   ```

4. **Start the application**
   ```bash
   npm start
   ```

1. **Install MongoDB**

   ```bash
   mongo
   use una_institute
   ```
